# dispute

争议人员管理前端项目，基于 React、TypeScript、Vite 和 Tailwind CSS 构建。

## 本地启动

前置要求：Node.js 18+

1. 安装依赖

```bash
npm install
```

2. 启动开发环境

```bash
npm run dev
```

3. 本地构建校验

```bash
npm run build
```
